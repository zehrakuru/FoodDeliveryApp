# FoodDeliveryApp

This is a food delivery app that users can order a food/meal which uses many features. Features are explained below. 

## Features That Is Used In This Project

1. Splash Screen
2. Onboarding (using ViewPager)
3. Recyclerview
4. Retrofit
5. Room Database
6. MVVM
7. Animation
8. Alert Dialog
9. Bottom Navigation
